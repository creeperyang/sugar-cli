Installation and Setup

Installation and Setup

通过npm安装即可，推荐全局安装npm i -g sugar-cli。

环境要求

  • NodeJS (v6.0.0及以上)
  • Windows, Linux, Unix 或 Mac OS X

快速开始

sugar-cli可以快速创建一个样板项目：

$ sugar init

你可以在当前目录看到创建的目录文件结构，其中包括配置文件sugar.config.js等等。

如果你希望指定目录，可以sugar init ./dir。

预览原型页面

sugar start [sugar.config.js]可以快速启动开发服务器，你可以通过浏览器实时访问。通常情况下，你都应该指定配置文件，配置文件中的配置将覆盖默认配置。

此外，sugar build可以生成静态网站。

Debug

sugar start --verbose可以在控制台看到详细输出，包括详细的处理过程和错误原因。
